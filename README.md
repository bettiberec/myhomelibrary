# My Home Library


My Home Library is a desktop application that lets you build, organize, and manage a virtual personal library.
The project was inspired by my sister, who always dreamed of creating her own home library â€” but never had enough physical space.
This application brings that idea to life by creating a fully customizable digital bookshelf where books can be added, visualized, rearranged.

The app features a realistic bookshelf, detailed book pages, JSON-based persistence, custom image handling and a drag&drop book rearrangement.

## Features

- Add new books with:
  - Title
  - Author
  - Year
  - Genre
  - Reading status
  - Summary/notes
  - Custom cover image
  - Custom spine image
- Edit existing books
- Delete books with confirmation
- Search & filter by:
  - Title
  - Author
  - Genre
  - Reading status
- Virtual Bookshelf
  - Wooden shelf background
  - Book spines rendered
  - Drag & Drop reordering (Edit Layout mode)
  - Saved layout persists across sessions
  - Animated hover effects
- Detailed Book View
  - Book-page styled display
  - Cover image
  - Notes
  - Rating
  - Navigation (Prev / Next)
  - Edit & Delete buttons
